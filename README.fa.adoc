مجموعه‌ای رو به رشد از کتابخانه‌های کوچک و کاربردی Rust برای توسعه‌دهندگان ایرانی.

🇬🇧 link:README.adoc[English version]

== کتابخانه‌ها

[cols="1,3", options="header"]
|===
| نام | توضیح

| link:crates/jalali[`jalali-calendar`]
| تقویم جلالی (شمسی): تاریخ، تاریخ-زمان، منطقه‌ی زمانی، قالب‌بندی strftime، تجزیه، محاسبات، و ابزارهای ارقام. (در کد به‌صورت `use jalali::*` ایمپورت می‌شود). به link:crates/jalali/README.adoc[راهنمای کتابخانه] مراجعه کنید.
|===

کتابخانه‌های بیشتر در راه است (ابزارهای متن فارسی، تبدیل عدد به حروف، کمک‌رسان‌های راست‌چین، اعتبارسنجی کد ملی و ...). از مشارکت شما استقبال می‌کنیم.

== مستندات

مستندات API به‌صورت خودکار از کامنت‌های `///` تولید می‌شود:

[source,bash]
----
cargo doc --workspace --no-deps --all-features --open
----
