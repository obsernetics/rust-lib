🇮🇷 link:README.fa.adoc[نسخه‌ی فارسی]

== Crates

[cols="1,3", options="header"]
|===
| Crate | Description

| link:crates/jalali[`jalali-calendar`]
| Jalali (Persian / Shamsi) calendar: dates, datetimes, timezones, strftime, parsing, arithmetic, and digit utilities. Imports as `use jalali::*`. See link:crates/jalali/README.adoc[crate readme].
|===

More crates are planned (Persian text utilities, number-to-words, RTL helpers, validators for national IDs, etc.). Contributions welcome.

== Documentation

API docs are generated from `///` doc comments via:

[source,bash]
----
cargo doc --workspace --no-deps --all-features --open
----

== Publishing

Each crate publishes independently to crates.io and is auto-built on docs.rs. See link:PUBLISHING.adoc[`PUBLISHING.adoc`] for the checklist.
